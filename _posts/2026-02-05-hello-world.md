## 為什麼選擇 Jekyll + Docker？

使用 Jekyll Gitbook 主題搭配 Docker 環境的優勢：

1. **隔離環境**：Docker 確保開發環境一致性
2. **即時預覽**：修改內容後立即看到效果
3. **版本控制**：所有內容都在 Git 管理下
4. **開源精神**：使用開放原始碼工具
